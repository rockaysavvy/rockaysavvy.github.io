The SWAT Pump is a unique uncommon shotgun. It consists of a modified Pump with a choke which provides significantly boosted performance. It is comparable to the Lupara Mk. III with hugely improved handling but notably worse reload speed and accuracy, rendering it an inferior choice of weapon.

Appearances

  • Some tier III SWAT officers carry Pumps.
  • Tier I shock troopers always use Pumps.

This is a fan site created and written by a single person. I love this game to pieces and enjoy sharing the world and mechanics of CBRC with others in the community.

Corrections

I thoroughly appreciate corrections to my work if they are clear and supported by evidence. Please submit corrections as issues on the repository. (The site is programmatically generated and relies on game assets, so I can’t publish the source or accept pull requests.) If you don’t include evidence of your claim—e.g., DVR or references to specific game assets—I will be disinclined to respond.

Fair Use

If you work for IGS/505: thank you so much for making this game. Please keep doing what you do. This website is noncommercial, utilizes only small portions of the game experience, doesn’t participate in the video game market, and exists solely to educate people about CBRC and encourage them to play it. If you wish to contact me, please file an issue on the repository using an official account.
